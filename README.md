## Featured Projects

### Meeting to Action AI
A functional product prototype that turns messy meeting notes into structured action briefs for product, design, and operations teams.

- Focus: AI-native workflow design, UX writing, product thinking, front-end prototyping
- Demo: https://saisha12345.github.io/meeting-to-action-ai-project/
- Repo: https://github.com/saisha12345/meeting-to-action-ai-project

### Twitter Storage & Retrieval System
A backend system built with Python, MySQL, Redis, and SQL to support tweet posting and home timeline retrieval at scale.

- Benchmarked 1M tweet insertions and 50K timeline requests
- Compared relational and in-memory storage tradeoffs
- Improved insertion throughput from approximately 3,906 to 5,574 calls per second using Redis

### MongoDB News Classification Data Pipeline
A document-store pipeline for transforming raw sarcasm headline data into structured MongoDB-ready documents.

- Cleaned malformed URLs and extracted source domains
- Designed nested headline records with derived text features
- Used aggregation pipelines to calculate sarcasm rates and keyword trends

### Aircraft Incident Data Quality & Analytics Database
A normalized MySQL analytics database for aircraft incident data.

- Designed a 3NF schema using statistical analysis and functional dependencies
- Built an R ETL pipeline for CSV preparation and loading
- Deployed a cloud-hosted MySQL database on Aiven with SSL
